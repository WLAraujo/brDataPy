# brDataPy
Repositório do projeto de código aberto BrDataPy, um pacote Python que visa conectar dados públicos do Brasil disponíveis em fontes diversas com usuários da linguagem Python via uma interface de pacote

O projeto visa disponibilizar em formatos diversos (csv, xlsx) dados de órgãos públicos das três esferas. O pacote serve como uma camada semântica e de governança que auxilia desenvolvedores e interessados em usar e consultar os dados a encontrar e recuperar de forma mais fácil. Assim, a principal proposta do projeto é ser um catálogo que fornece os meios de acesso a dados públicos que estão amplamente pulverizados em vários portais.

O grande propósito do projeto é facilitar o acesso de usuários Python em geral a dados públicos brasileiros permitindo criação de valor público com eles.

A organização do repositório seguirá as melhores práticas de repositórios de pacotes que usam Poetry, uma ferramenta Pyhton que facilita a criação de pacotes. Já existe uma estrutura pré-montada mas as funções serão desenvolvidas especificamente para cada fonte de dados.
